Angular 16 provide the ability to bind route parameters to the corresponding component’s inputs. Now we can pass the below data to a routing component’s inputs:

  • Route data — resolvers and data properties
  • Path parameters
  • Query parameters

First of all, we need to activate it. In a standalone application, this is done by adding withComponentInputBinding() to application's bootstrap configuration (watch main.ts and app.config.ts files).



Once this is done, we can use @Input to retrieve wanted data into component. Please watch code both from routes definition (app.routes.ts) and router-child-component for more details.
